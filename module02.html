Core Concept
This module teaches you how to take a real jewellery piece from a supplier site and generate a professional model shot using the Clone skill and Higgsfield.

By the end of this module you will know how to do this for any piece. A ring. A necklace. An earring. Any product. Any look. Every time.
The Matching Rule
Your reference image must match the type of jewellery you want to shoot.

If you want to generate a necklace shot find a reference of a model wearing a necklace. If you want a ring shot find a reference with a ring. If you want an earring shot find a reference with an earring.

The Clone skill recreates the position and placement from the reference. If the position already exists in the image the jewellery drops in cleanly. If it does not exist the result will not be accurate.

Match the reference to the piece every single time.

The System
  • 1Go to any jewellery supplier site. If you are a founding member use the supplier list in Bonus 01
  • 2Pick one piece. A ring, a necklace, or an earring
  • 3Find a reference image that matches that piece type
  • 4Open your R8 Studio Skills project in Claude
  • 5Type clone then upload your reference image
  • 6Take the output prompt into Higgsfield
  • 7Replace the jewellery
